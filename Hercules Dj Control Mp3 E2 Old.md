# Hercules DJ Control MP3 e2

## Mapping for Mixxx 1.11 and below

The Hercules MP3 e2 mapping for Mixxx is integrated in Mixxx so you
don't have to download or install nothing.

[[/media/hercules_mappa.png|]]
